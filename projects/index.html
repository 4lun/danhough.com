Like many developers, I tend to have at least a couple of projects going at any one time. In October 2013, I left my full-time job at Huddle in order to pursue a more flexible lifestyle working on these projects and my various hobbies more seriously. Below is a list of various projects I've been involved in. Some of them are ongoing, others are completed or came to a natural end. Lololol.

Interfake

Super-simple JSON API Prototyping

Interfake came about when a colleague complained about having to constantly create dummy servers while testing an iOS app. Soon, it became a lot more flexible and powerful. It's commonly used in creating API prototypes for nascent projects and for creating mock/dummy APIs for testing. Plus, it's open-source and I'm always looking for new contributors. It's written in NodeJS.

See more on the GitHub repo.

StreetScout

Web and iOS Routing Application

StreetScout is a routing application for web and iOS which allows the user to find and visualise places on their routes which match a given query, such as coffee shops, pubs or grocery shops. Using Google Maps and Google Places data.

See more on the StreetScout website.

SkiFree.js

Web & Mobile Web Game

SkiFree was a popular PC Game back in the 90s which came with Microsoft Windows. I created a JavaScript port of it in early 2013, and it became my highest-rated Hacker News submission, and first-ever frontpage entry, too.

Play the game now on GitHub.

Code Club

Volunteering with kids

Code Club is a nationwide network of weekly after-school clubs run by volunteers to help British Primary School kids to program by teaching them how to make simple games. I volunteer to run one of these clubs. It's great fun and very rewarding. If you're at all experienced with programming, I'd highly recommend it.

Go to the Code Club website.
